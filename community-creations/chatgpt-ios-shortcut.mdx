A plugin for Supernotes that utilizes the ChatGPT app and API to automatically generate a title for quick notes.

## Features

* Asks for text input and sends it to ChatGPT with a prompt to generate a title
* Allows choice between GPT-4 and GPT-3.5 models
* Compatible with Supernotes API
* Optionally adds timestamp to title

## Setup

1. Create a new card in Supernotes and grab its ID
2. Add the ID to the 'parent_ids' section in the plugin setup
3. Configure the plugin to use the ChatGPT app and API
